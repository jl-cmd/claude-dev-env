# build_goal_constants

Named constants package for the `build-goal` skill's `scripts/write_goal_pair.py` CLI.

## Key files

| File | Purpose |
|---|---|
| `__init__.py` | Package marker |
| `write_goal_pair_constants.py` | Run-directory and template file names, the packet JSON schema keys, the template placeholders, bullet and table row formats, the stdout path-print prefixes, exit codes, and every stderr error message |
